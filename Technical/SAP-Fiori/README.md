# SAP Fiori

> **SAP Fiori** is SAP's modern user experience (UX) design system that provides role-based, responsive, and intuitive business applications. It replaces traditional SAP GUI screens with web and mobile-friendly applications, improving productivity and user satisfaction across SAP solutions.

# Overview

SAP Fiori provides a modern, consumer-grade user experience for SAP applications. It delivers simple, role-based applications that can be accessed from desktops, tablets, and smartphones.

Organizations use SAP Fiori to:

- Modernize SAP applications
- Improve user experience
- Increase employee productivity
- Enable mobile access
- Simplify business processes
- Reduce training effort
- Standardize application design

---

# Core Concepts

SAP Fiori is built around three core principles:

- **Role-Based** – Users only see applications relevant to their job.
- **Responsive** – Applications work across desktop, tablet, and mobile devices.
- **Simple** – Clean, intuitive interfaces with fewer clicks.

# Real-World Example

A sales manager wants to approve customer quotations while traveling.

1. A Fiori application is developed using SAP UI5.
2. Backend data is exposed through OData services.
3. The app is deployed to the SAP Fiori Launchpad.
4. The manager accesses the application from a mobile device.
5. Approvals are processed instantly without using SAP GUI.
