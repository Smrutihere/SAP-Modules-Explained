# SAP ABAP (Advanced Business Application Programming)

> **SAP ABAP (Advanced Business Application Programming)** is SAP's proprietary programming language used to develop, customize, and enhance SAP applications. ABAP developers build reports, interfaces, forms, workflows, APIs, enhancements, and business logic for SAP ECC and SAP S/4HANA systems.

# Overview

SAP ABAP is the backbone of SAP technical development. It enables developers to create custom applications, reports, interfaces, enhancements, and integrations that extend standard SAP functionality.

Organizations use ABAP to:

- Develop custom reports
- Build interfaces
- Create custom transactions
- Develop APIs
- Build Fiori backend services
- Create Smart Forms and Adobe Forms
- Develop enhancements
- Customize SAP business processes

# Interview Questions

## Recruiter Screening Questions

1. Which SAP versions have you worked on (ECC, S/4HANA)?
2. How many years of ABAP development experience do you have?
3. Have you worked on Object-Oriented ABAP?
4. Have you developed CDS Views?
5. Have you worked with OData Services?
6. Have you developed Smart Forms or Adobe Forms?
7. Have you worked on Interfaces using IDocs or BAPIs?
8. Have you worked on SAP Fiori backend development?

# Real-World Example

A manufacturing company needs a custom report showing delayed purchase orders.

1. The functional consultant gathers business requirements.
2. The ABAP developer designs the report.
3. Data is retrieved from SAP tables.
4. An ALV report is created with filters and sorting.
5. Users execute the report using a custom transaction code.
6. The report improves visibility into delayed procurement activities.
