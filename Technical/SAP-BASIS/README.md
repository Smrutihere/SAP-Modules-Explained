# Overview

SAP BASIS acts as the bridge between the SAP application and the underlying operating system, database, and hardware. BASIS consultants ensure SAP systems are secure, available, optimized, and running efficiently.

Organizations use SAP BASIS to:

- Install SAP systems
- Configure SAP landscapes
- Monitor system performance
- Manage transports
- Administer users and roles
- Perform upgrades and migrations
- Manage backups and disaster recovery
- Ensure system availability

# Interview Questions

## Recruiter Screening Questions

1. Which SAP versions have you supported (ECC, S/4HANA)?
2. Have you performed SAP installations?
3. Have you managed system refreshes or client copies?
4. Have you worked with SAP HANA databases?
5. Have you handled kernel upgrades?
6. Have you worked on cloud environments such as AWS, Azure, or GCP?
7. Have you administered SAP Fiori or Gateway?

## Technical Interview Questions

1. What is SAP BASIS?
2. Explain the SAP transport process.
3. What is STMS?
4. What are SAP Work Processes?
5. Explain Client Copy.
6. What is a System Refresh?
7. How do you perform a Kernel Upgrade?
8. What is SAP Host Agent?
9. Explain SAP Profiles.
10. How do you troubleshoot performance issues?

# Real-World Example

A company is upgrading from SAP ECC to SAP S/4HANA.

1. BASIS installs the new SAP landscape.
2. SAP HANA database is configured.
3. System parameters are optimized.
4. Transport routes are created.
5. User roles are migrated.
6. Kernel patches are applied.
7. System performance is monitored after go-live.
