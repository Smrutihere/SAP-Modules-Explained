# SAP UI5

> **SAP UI5** is SAP's JavaScript-based front-end framework for developing responsive, enterprise-grade web applications. It is the foundation of SAP Fiori applications and enables developers to build modern user interfaces that interact with SAP backend systems using OData services and REST APIs.

# Overview

SAP UI5 is an HTML5, CSS, and JavaScript framework developed by SAP for building responsive business applications.

It follows the MVC (Model-View-Controller) architecture and is widely used for developing SAP Fiori applications across desktop, tablet, and mobile devices.

Organizations use SAP UI5 to:

- Develop SAP Fiori applications
- Build responsive web applications
- Create enterprise dashboards
- Consume OData services
- Develop custom SAP interfaces
- Enhance user experience
