# SAP Analytics Cloud (SAC)

> **SAP Analytics Cloud (SAC)** is SAP's cloud-based analytics platform that combines Business Intelligence (BI), planning, predictive analytics, and augmented analytics into a single solution. It enables organizations to create interactive dashboards, perform financial planning, forecast business outcomes, and make data-driven decisions in real time.

# Overview

SAP Analytics Cloud is a Software-as-a-Service (SaaS) solution that provides analytics, planning, and predictive capabilities through a web-based interface.

Organizations use SAP Analytics Cloud to:

- Build interactive dashboards
- Perform business intelligence reporting
- Create financial and operational plans
- Forecast future business performance
- Analyze enterprise data
- Enable self-service analytics
- Support executive decision-making

# Real-World Example

A retail company wants to monitor sales performance across multiple regions.

1. Sales data is retrieved from SAP S/4HANA.
2. SAP Analytics Cloud connects using a Live Connection.
3. Interactive dashboards display sales by region, product, and time.
4. Executives use Smart Insights to identify sales trends.
5. Finance teams build forecasts using Planning Models.
6. Management makes faster, data-driven business decisions.
