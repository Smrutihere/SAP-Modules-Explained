# SAP GTS (Global Trade Services)

> **SAP GTS (Global Trade Services)** is an SAP solution that helps organizations manage international trade processes, customs compliance, export controls, import management, and global trade regulations. It enables businesses to conduct cross-border trade while complying with international laws and minimizing trade risks.

# Overview

SAP GTS helps organizations automate and streamline international trade operations while ensuring compliance with customs regulations, export controls, sanctions, and trade agreements.

Organizations use SAP GTS to:

- Manage international trade
- Ensure customs compliance
- Screen business partners
- Classify products
- Manage import and export declarations
- Calculate duties and taxes
- Reduce trade compliance risks

# Real-World Example

A manufacturing company exports machinery from Germany to the United States.

1. A sales order is created in SAP SD.
2. SAP GTS performs Sanctioned Party List (SPL) screening.
3. The product is classified using the correct tariff code.
4. Export license requirements are validated.
5. Customs documentation is generated.
6. SAP TM plans transportation.
7. Goods are shipped after successful compliance checks.
8. Duty and tax information is posted to SAP FI.
