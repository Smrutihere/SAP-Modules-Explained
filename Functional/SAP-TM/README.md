# SAP TM (Transportation Management)

> **SAP TM (Transportation Management)** is a core SAP Functional module that helps organizations plan, execute, monitor, and optimize the transportation of goods across the supply chain. It supports freight planning, carrier selection, route optimization, freight costing, shipment execution, and transportation analytics.

# Overview

SAP TM enables organizations to efficiently manage domestic and international transportation by optimizing shipment planning, reducing freight costs, and improving delivery performance.

Organizations use SAP TM to:

- Plan transportation
- Optimize routes
- Select carriers
- Calculate freight costs
- Track shipments
- Improve delivery performance
- Reduce logistics costs

# Real-World Example

A retail company needs to deliver products from multiple warehouses to stores across the country.

1. Sales orders are created in SAP SD.
2. SAP TM groups deliveries into Freight Units.
3. The system selects the most cost-effective carrier.
4. Routes are optimized.
5. Freight Orders are created.
6. Shipments are tracked until delivery.
7. Freight invoices are settled and posted to SAP FI.
