# SAP SD (Sales and Distribution)

> **SAP SD (Sales and Distribution)** is a core SAP Functional module that manages the complete Order-to-Cash (O2C) process. It helps organizations handle customer inquiries, sales orders, shipping, billing, pricing, and customer payments efficiently.

# Overview

SAP SD is responsible for managing sales, customer relationships, order processing, shipping, billing, and invoicing. It ensures products are delivered to customers accurately and payments are collected efficiently.

Organizations use SAP SD to:

- Process customer inquiries
- Create quotations
- Manage sales orders
- Deliver products
- Generate invoices
- Handle pricing and discounts
- Process returns
- Maintain customer master data

# Real-World Example

An electronics company receives an order for 100 laptops.

1. Sales creates a quotation.
2. Customer approves the quotation.
3. Sales Order is created.
4. ATP confirms stock availability.
5. Warehouse prepares shipment.
6. Goods are delivered.
7. Billing generates the customer invoice.
8. SAP FI records the revenue and payment.
