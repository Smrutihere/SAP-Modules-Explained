# SAP PS (Project System)

> **SAP PS (Project System)** is a core SAP Functional module used to plan, execute, monitor, and control projects. It enables organizations to manage project structures, budgets, costs, resources, schedules, and project execution throughout the entire project lifecycle.

# Overview

SAP PS helps organizations manage simple and complex projects by providing structured planning, budgeting, scheduling, resource allocation, and cost tracking. It is widely used in industries where projects are central to business operations, such as construction, engineering, manufacturing, aerospace, utilities, and IT services.

Organizations use SAP PS to:

- Plan and manage projects
- Create Work Breakdown Structures (WBS)
- Schedule project activities
- Allocate resources
- Track project costs and budgets
- Monitor project progress
- Analyze project profitability

# Real-World Example

A construction company wins a contract to build a commercial office complex.

1. A Project Definition is created.
2. The project is divided into WBS elements such as Design, Foundation, Electrical, Plumbing, and Finishing.
3. Networks and Activities are created for each phase.
4. Budgets are assigned to WBS elements.
5. Materials are procured through SAP MM.
6. Labor and project costs are tracked through SAP CO.
7. Project progress is monitored using milestones.
8. After completion, project costs are settled to SAP FI.
